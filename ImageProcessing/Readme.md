# Лабораторная работа №2
Java-приложение с графическим интерфейсом предоставляет комплекс инструментов для обработки изображений, включая обнаружение особенностей и адаптивную бинаризацию.

### Функциональность

#### Основные возможности:
1. **Загрузка изображений** - поддержка различных форматов через ImageIO
2. **Обнаружение точек** - выделение изолированных точек с помощью лапласиана
3. **Обнаружение линий** - выявление горизонтальных, вертикальных и диагональных линий
4. **Детектирование перепадов яркости** - вычисление градиентов с помощью оператора Собеля
5. **Адаптивная бинаризация** - два метода локального порогового преобразования:
   - На основе среднего значения в окрестности
   - На основе медианного значения в окрестности

#### Интерфейс:
- Интуитивный графический интерфейс на Java Swing
- Параллельное отображение оригинального и обработанного изображений
- Автоматическое масштабирование изображений для удобного просмотра
- Информативные подписи с размерами изображений
- Доступные кнопки для всех функций обработки

### Используемые технологии
- **Java Swing** - для создания графического интерфейса
- **Java AWT** - для работы с изображениями и графикой
- **ImageIO** - для загрузки и чтения файлов изображений
- **BufferedImage** - для обработки пиксельных данных
- **Матричные операции свертки** - для реализации фильтров

### Использование

#### Пошаговая инструкция:
1. **Загрузка изображения:**
   - Нажмите кнопку "Загрузить изображение"
   - Выберите файл изображения в диалоговом окне
   - Оригинальное изображение появится в левой части окна

2. **Обнаружение точек:**
   - Нажмите кнопку "Обнаружение точек"
   - Точки будут выделены красным цветом на изображении

3. **Обнаружение линий:**
   - Нажмите кнопку "Обнаружение линий"
   - Линии будут выделены синим цветом

4. **Детектирование градиентов:**
   - Нажмите кнопку "Обнаружение перепадов яркости"
   - Отобразится карта градиентов с нормализацией интенсивности

5. **Локальная бинаризация:**
   - Выберите один из двух методов пороговой обработки
   - Получите бинарное изображение с адаптивным порогом

### Особенности работы
- Программа автоматически конвертирует цветные изображения в градации серого для обработки
- Все операции применяются к текущему загруженному изображению
- Можно последовательно применять разные методы обработки
- Реализовано корректное отображение как цветных, так и бинарных изображений

### Основные алгоритмы

#### Обнаружение точек (Лапласиан)
- Применяется дискретный лапласиан с ядром 3×3
- Вычисляется отклик фильтра для каждого пикселя
- Точки определяются по превышению порогового значения отклика
- Обнаруженные точки отмечаются красным цветом

#### Обнаружение линий
- Используются четыре направленных ядра:
  - Горизонтальное
  - Вертикальное 
  - Диагональное (два направления)
- Выбирается максимальный отклик среди всех направлений
- Линии выделяются синим цветом при превышении порога

#### Детектирование градиентов (Оператор Собеля)
- Вычисляются градиенты по осям X и Y
- Рассчитывается магнитуда градиента
- Производится нормализация значений для визуализации

#### Локальная пороговая обработка
1. **Метод среднего значения:**
   - Вычисляется средняя яркость в окрестности
   - Порог = среднее - константа

2. **Медианный метод:**
   - Вычисляется медиана яркостей в окрестности
   - Порог = медиана - константа
