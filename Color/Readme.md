# Лабораторная работа: Конвертер цветовых моделей

## Цель работы
Разработка графического приложения для конвертации цветов между различными цветовыми моделями (RGB, CMYK, HSV).
## Описание проекта

Программа **ColorModelConverter** представляет собой Java-приложение с графическим интерфейсом, позволяющее конвертировать цвета между тремя основными цветовыми моделями:
- **RGB** (Red, Green, Blue) - аддитивная модель для электронных дисплеев
- **CMYK** (Cyan, Magenta, Yellow, Key/Black) - субтрактивная модель для печати
- **HSV** (Hue, Saturation, Value) - модель, основанная на восприятии цвета человеком

## Функциональность

### Основные возможности:
- **Конвертация в реальном времени** между всеми цветовыми моделями
- **Визуальное представление** текущего цвета в панели предпросмотра
- **Текстовое представление** цвета в HEX-формате
- **Интуитивное управление** через слайдеры и текстовые поля
- **Автоматическая синхронизация** - изменение в одной модели мгновенно отражается в других

### Элементы управления:
- **Слайдеры** для плавного изменения значений компонентов
- **Текстовые поля** для точного ввода числовых значений
- **Панель предпросмотра** с визуализацией текущего цвета
- **HEX-поле** с текстовым представлением цвета

## Архитектура приложения

### Основные классы:

#### `RGBPane.java`
- Панель управления моделью RGB
- Содержит слайдеры и поля для компонентов R, G, B (0-255)
- Обеспечивает ввод и отображение значений

#### `CMYKPane.java`
- Панель управления моделью CMYK  
- Содержит слайдеры и поля для компонентов C, M, Y, K (0-100%)
- Преобразует проценты в дробные значения (0.0-1.0)

#### `HSVPane.java`
- Панель управления моделью HSV
- Содержит слайдеры и поля для H (0-360°), S (0-100%), V (0-100%)
- Обрабатывает угловые и процентные значения

#### `ColorConverter.java`
- Математический модуль конвертации
- Содержит алгоритмы преобразования между моделями
- Обеспечивает точность вычислений и корректные диапазоны значений

#### `ColorController.java`
- Центральный контроллер приложения
- Координирует взаимодействие между всеми компонентами
- Обрабатывает события изменения значений
- Управляет синхронизацией данных между моделями
- Защита от циклических обновлений через флаг `updating`

#### `Main.java`
- Точка входа в приложение
- Настраивает графический интерфейс
- Располагает компоненты в окне приложения

## Интерфейс пользователя

### Компоновка:
- **Вертикальное расположение** панелей цветовых моделей
- **Горизонтальное расположение** элементов внутри каждой панели
- **Правая панель** с предпросмотром и дополнительной информацией

### Элементы управления в каждой панели:
- Метка компонента (R, G, B / C, M, Y, K / H, S, V)
- Слайдер с делениями для визуального управления
- Текстовое поле для точного числового ввода

## Особенности реализации

### Синхронизация данных:
- Использование флага `updating` для предотвращения бесконечных циклов
- Последовательное обновление всех моделей при изменении любой компоненты
- Обработка событий от слайдеров и текстовых полей

### Обработка ошибок:
- Защита от некорректного ввода в текстовых полях
- Ограничение значений в допустимых диапазонах
- Игнорирование нечисловых входных данных

### Визуальные улучшения:
- Широкие слайдеры для удобного управления
- Горизонтальная компоновка элементов
- Четкие границы и метки для каждой цветовой модели
- Информационное сообщение о возможной погрешности конвертации

## Использование

1. **Изменение цвета** через любую из панелей (RGB, CMYK, HSV)
2. **Наблюдение** автоматического обновления во всех моделях
3. **Просмотр** результата в панели предпросмотра
4. **Копирование** HEX-значения для использования в других приложениях
