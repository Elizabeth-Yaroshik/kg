# Лабораторная работа №1
Программа ColorModelConverter представляет собой графическое приложение на Java для конвертации цветов между тремя популярными цветовыми моделями: RGB, CMYK и HSV. Приложение предоставляет интуитивно понятный интерфейс для просмотра и редактирования цветов в различных моделях с возможностью выбора цвета из палитры.

## Функциональность
### Конвертация между цветовыми моделями:
* RGB (Red, Green, Blue)
* CMYK (Cyan, Magenta, Yellow, Key/Black)
* HSV (Hue, Saturation, Value)

### Визуальное представление:
* Панель предпросмотра цвета с отображением HEX-значения
* Слайдеры для регулировки компонентов цвета
* Спиннеры для точного ввода значений
* Текстовые поля для точного ввода значений
* Информационная панель с детальными данными о цвете
* Кнопка выбора цвета из системной палитры

### Синхронизация данных:
* Автоматическое обновление всех моделей при изменении любой компоненты
* Защита от циклических обновлений через флаг `updating`
* Связь между слайдерами и спиннерами

## Архитектура приложения

### Основные классы:

**ColorPanel** - универсальная панель для управления компонентами цвета:
* Содержит слайдеры и спиннеры для каждой компоненты
* Поддерживает различные диапазоны значений
* Обеспечивает синхронизацию между слайдерами и спиннерами

**ColorPreviewPanel** - панель предпросмотра цвета:
* Отображает текущий цвет
* Показывает HEX-значение с автоматическим подбором контрастного текста
* Имеет фиксированный размер для стабильного отображения

**ColorController** - центральный контроллер управления:
* Координирует взаимодействие между всеми компонентами
* Обрабатывает события изменения значений
* Выполняет конвертацию между цветовыми моделями
* Обновляет информационную панель

**ConversionUtils** - утилиты конвертации:
* `rgbToCmyk()` - преобразование RGB в CMYK
* `cmykToRgb()` - преобразование CMYK в RGB
* `rgbToHsv()` - преобразование RGB в HSV
* `hsvToRgb()` - преобразование HSV в RGB

**Модели данных:**
* `RGBColor` - модель цвета RGB (0-255)
* `CMYKColor` - модель цвета CMYK (0.0-1.0)
* `HSVColor` - модель цвета HSV (0-360°, 0.0-1.0, 0.0-1.0)

## Использование
### Основные элементы управления:

**RGB панель:**
* Слайдеры R, G, B (0-255)
* Спиннеры для точного ввода значений

**CMYK панель:**
* Слайдеры C, M, Y, K (0-100%)
* Спиннеры для точного ввода процентов

**HSV панель:**
* Слайдер Hue (0-360°)
* Слайдеры Saturation, Value (0-100%)
* Спиннеры для точного ввода

**Панель предпросмотра:**
* Визуальное отображение текущего цвета
* HEX-представление цвета
* Автоматический контраст текста

**Информационная панель:**
* Детальные значения CMYK, RGB, HSV
* Дополнительная информация (яркость, luminance)
* HEX-код цвета
